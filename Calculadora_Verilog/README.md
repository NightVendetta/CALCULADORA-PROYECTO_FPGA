# Módulos de la Calculadora

Esta carpeta contiene los diferentes módulos que componen la calculadora digital. Cada subcarpeta corresponde a un módulo específico y contiene los archivos fuente en Verilog, así como la documentación relacionada.

## Lista de Módulos

### [Multiplicación](./multiplicacion/)
Contiene el código del multiplicador que implementa el algoritmo de desplazamiento y suma.

### [División](./division/)
Incluye el código del divisor que utiliza el algoritmo de restas sucesivas con desplazamiento.

### [Raíz Cuadrada](./raiz_cuadrada/)
Alberga el código para el cálculo de raíz cuadrada mediante el método de desplazamiento y resta.

### [Binario a BCD](./bin2bcd/)
Contiene el conversor de binario a BCD usando el algoritmo de desplazamiento y suma de 3.

### [BCD a Binario](./bcd2bin/)
Incluye el conversor de BCD a binario, que es el inverso del algoritmo anterior.

## Estructura de Cada Módulo

Cada subcarpeta de módulo contiene:
- Archivos fuente en Verilog (`.v`)
- Testbenches (`.v`)
- Perifericos
